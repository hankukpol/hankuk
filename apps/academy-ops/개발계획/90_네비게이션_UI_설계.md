## 2. 상단 탭 (Top Navigation) — 7탭 구조

### 2-1. 탭 구성

| 탭 | 아이콘 | 접근 권한 | URL 기준 |
|---|---|---|---|
| 대시보드 | 🏠 | 전체 | `/admin` |
| 회원·수강 | 👥 | ACADEMIC_ADMIN 이상 | `/admin/students/*`, `/admin/enrollments/*` |
| 수납·결제 | 💳 | MANAGER 이상 | `/admin/payments/*` |
| 성적·출결 | 📊 | TEACHER 이상 | `/admin/scores/*`, `/admin/attendance/*` |
| 시설·포인트 | 🏢 | ACADEMIC_ADMIN 이상 | `/admin/facilities/*`, `/admin/points/*` |
| 분석·보고 | 📈 | DIRECTOR 이상 | `/admin/analytics/*`, `/admin/reports/*` |
| 설정 | ⚙️ | DIRECTOR 이상 | `/admin/settings/*`, `/admin/staff/*` |

> SUPER_ADMIN은 별도로 `/admin/super/*` 접근 가능 (탭에 표시 안 됨 — 설정 탭 안에 포함)

### 2-2. 탭 UI 스타일

```
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│  ◈ AcadPro  대시보드  회원·수강  수납·결제  성적·출결  시설·포인트  분석·보고  설정  │
│             ─────────                                                    │
│             현재탭 밑줄           [한국경찰학원 ▼]   [🔔2]  [이원장 ▾]   │
└──────────────────────────────────────────────────────────────────────────┘

스타일:
- 배경: forest(#1F4D3A) 또는 ink(#111827) 다크 바
- 일반 탭: 흰색 60% 투명도
- 선택된 탭: 흰색 100% + 하단 ember(#C55A11) 2px 밑줄
- 높이: 56px / 탭 간격: px-4

[지점명 ▼] 드롭다운:
- SUPER_ADMIN만 표시 (ember 색상 강조)
- 일반 관리자: 소속 지점명만 텍스트로 표시 (클릭 불가)
- 드롭다운 항목: [전체 보기] / [한국경찰학원] / [한국소방학원] / ...
```

### 2-3. 알림 아이콘

```
🔔 숫자 뱃지 = 처리 필요 항목
  - 미승인 결석계
  - 미납 독촉 예정
  - 환불 승인 대기
  - 면담 예약

클릭 시: 드롭다운으로 항목 목록 표시 (지점 필터 적용)
```

## 4. 학생 상세 페이지 (특수 레이아웃)

학생 개별 페이지(`/admin/students/[examNumber]`)는 **탭 UI**를 사용한다.
좌측 사이드바 없이 콘텐츠 내 탭으로 전환. 탭은 12개 → **6개로 통합**.

```
┌──────────────────────────────────────────────────────────────────────────┐
│  TOP NAV: [회원·수강 탭 선택됨]                                          │
├──────────┬───────────────────────────────────────────────────────────────┤
│  회원·   │  ← 학생 목록으로 돌아가기                                    │
│  수강    │                                                               │
│  사이드  │  홍길동 (학번: 81697)  공채·52기  [재원]  [수정]             │
│  바      │  010-1234-5678  |  포인트 3,200P  |  등록일 2024-01-15        │
│          │                                                               │
│  (기존   │  [기본·수강]  [수납·결제]  [성적·출결]  [상담]  [시설]  [문서]│
│  메뉴    │  ─────────────                                               │
│  유지)   │                                                               │
│          │  탭 콘텐츠 영역...                                            │
└──────────┴───────────────────────────────────────────────────────────────┘
```

**학생 상세 탭 6개 구성:**

| 탭 | 포함 내용 |
|---|---|
| 기본·수강 | 기본정보 + 수강 이력 + 반 편성 정보 |
| 수납·결제 | 수납 이력 + 미납 현황 + 환불 이력 |
| 성적·출결 | 성적 분석 + 출결 캘린더 + 결석계 |
| 상담 | 상담 기록 + 면담 이력 |
| 시설 | 사물함 배정 + 스터디룸 예약 + 포인트 이력 |
| 문서 | 수강계약서 + 수강증 + 각종 증명서 |

## 7. 기존 페이지 마이그레이션 전략

기존 `/admin/*` 페이지들은 URL 변경 없이 Route Group으로 재분류만 하면 된다.

```
현재: web/src/app/admin/analytics/page.tsx
이동: web/src/app/admin/(analytics)/analytics/page.tsx
URL:  /admin/analytics (동일 — 변경 없음)
```

**주의**: 기존 성적관리 탭에 있던 analytics/* 경로들은 이제 "분석·보고" 탭으로 이동.
URL은 변경되지 않지만 네비게이션 탭 매핑이 달라짐.
