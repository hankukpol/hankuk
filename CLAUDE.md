## Claude 전용 지침

### 메모리
- 세션 간 기억이 필요한 정보는 `C:\Users\kunry\.claude\projects\d--hankuk\memory\MEMORY.md`에 저장한다.
- 저장 전 기존 메모리와 중복되지 않는지 확인한다.

### 작업 방식
- 코드를 수정하기 전에 반드시 해당 파일을 먼저 읽는다.
- 새 파일 생성보다 기존 파일 수정을 우선한다.
- 큰 변경은 계획을 먼저 제시하고 사용자 승인 후 진행한다.
- 앱 작업 시 해당 앱의 패키지 매니저를 확인하고 사용한다 (현재 3개 앱 모두 npm).

### Supabase 작업 시 주의사항
- 모든 앱은 단일 Supabase 프로젝트를 공유한다 (AGENTS.md 참조).
- 새 앱 추가 시 Supabase 프로젝트를 새로 만들지 않고, 기존 프로젝트에 스키마를 추가한다.
- 공통 데이터(회원, 인증)는 `public` 스키마, 앱 전용 데이터는 앱별 스키마에 둔다.
- 기존 5개 Supabase 프로젝트는 마이그레이션 완료 후 30일 유지 후 삭제 예정이므로, 삭제하지 않는다.
