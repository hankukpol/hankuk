## 📊 DB 모델 전체 목록

### 기존 (재활용)
- `Student` — 수험번호 기반 학생 마스터
- `AdminUser` — 관리자/직원 계정
- `ExamSession` — 시험 회차
- `Score` — 성적
- `Attendance` — 출결
- `AbsenceNote` — 사유서
- `CounselingRecord` — 면담 기록
- `PointLog` — 포인트 이력

### 신규 추가 (마스터플랜)
- `MemberProfile` — 회원 확장 정보 (Student 1:1)
- `ConsultationRecord` — 신규 방문 상담
- `CourseEnrollment` — 수강 등록
- `Course` — 강좌 마스터
- `Payment` — 수납 원장
- `Refund` — 환불
- `PosTransaction` — 단과 POS 거래
- `Locker` / `LockerAssignment` — 사물함
- `StudyRoom` / `StudyRoomReservation` — 스터디룸
- `Book` / `BookLoan` — 교재 대여
- `PointLedger` — 포인트 원장 (확장)
- `ExternalExam` / `ExternalExamScore` — 외부 모의고사

### 신규 추가 (수납결제 PRD)
- `PaymentItem` — 수납 항목 상세
- `Installment` — 분할납부 스케줄
- `Textbook` — 교재 판매 마스터
- `TextbookBundle` / `TextbookBundleItem` — 교재 세트
- `TextbookStockLog` — 교재 재고 이력
- `CardPayment` — 카드 결제 상세 (VAN 연동)
- `CardTerminalSettings` — VAN 단말기 설정
- `CashReceipt` — 현금영수증
- `DailySettlement` — 일계표
- `LeaveRecord` — 휴원 기록
