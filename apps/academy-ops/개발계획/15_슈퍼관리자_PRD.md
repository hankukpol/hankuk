# PRD: 슈퍼관리자 전용 기능

**작성일**: 2026-03-21
**위치**: `/admin/super/*` (SUPER_ADMIN 역할 전용)
**목적**: 복수 지점을 총괄 관리하는 슈퍼관리자 전용 기능 정의

---

## 설계 원칙

> **SUPER_ADMIN만** 이 섹션의 페이지에 접근할 수 있다.
> 다른 역할(DIRECTOR 포함)은 접근 시 403 Forbidden 반환.
>
> SUPER_ADMIN은 코드에서 단순 최고 권한자가 아니라
> **전 지점 데이터와 설정에 접근하는 별도 계층**이다.

---

## 1. 지점 관리 (`/admin/super/academies`)

### 1-1. 전체 지점 목록

```
┌──────────────────────────────────────────────────────────┐
│  전체 지점 관리                           [+ 신규 지점]   │
├──────┬────────────────┬──────────┬────────┬──────────────┤
│ ID   │ 지점명         │ 유형     │ 학생수 │ 상태         │
├──────┼────────────────┼──────────┼────────┼──────────────┤
│  1   │ 한국경찰학원   │ POLICE   │  243명 │ ✅ 활성      │
│  2   │ 한국소방학원   │ FIRE     │   87명 │ ✅ 활성      │
│  3   │ (준비 중)      │ OTHER    │    0명 │ ⏸ 비활성    │
└──────┴────────────────┴──────────┴────────┴──────────────┘
```

**기능:**
- 지점 활성화/비활성화 토글
- 지점별 기본 설정 보기 (→ `/admin/settings/academy` with academyId)
- 신규 지점 생성 (→ `/admin/super/academies/new`)

### 1-2. 신규 지점 등록 폼 (`/admin/super/academies/new`)

```
┌──────────────────────────────────────────────┐
│  신규 지점 등록                               │
├──────────────────────────────────────────────┤
│  지점 코드 (영문)   [ gyeongchal         ]   │
│  학원명            [ 한국경찰학원        ]   │
│  지점 유형         [POLICE ▼]                │
│    POLICE         경찰 공무원 학원            │
│    FIRE           소방 공무원 학원            │
│    CIVIL_SERVICE  일반 공무원 학원            │
│    OTHER          기타                        │
├──────────────────────────────────────────────┤
│  [취소]                          [지점 생성] │
└──────────────────────────────────────────────┘
```

**생성 시 자동 처리:**
1. `academies` 테이블에 신규 레코드 생성
2. `academy_settings` 기본값 레코드 자동 생성
3. `exam_subjects` 지점 유형별 기본 과목 자동 삽입
   - POLICE: 헌법, 형사법, 경찰학개론, 형사소송법, 범죄학
   - FIRE: 소방학개론, 소방관계법규, 소방시설론, 위험물론
   - CIVIL_SERVICE: 국어, 영어, 한국사, 행정법, 행정학

## 4. 지점 전환 UI (헤더 드롭다운)

**위치**: 상단 TOP NAV 오른쪽 (SUPER_ADMIN 로그인 시에만 표시)

```
┌──────────────────────────────────────────────────────────┐
│ ... 메뉴 탭 ...  [한국경찰학원 ▼]  [🔔2]  [SUPER_ADMIN ▾]│
│                  ─────────────────                        │
│                  ✓ 한국경찰학원                           │
│                    한국소방학원                            │
│                  ─────────────────                        │
│                    전체 보기 모드                          │
└──────────────────────────────────────────────────────────┘
```

**동작:**
1. 지점 선택 → `POST /api/admin/switch-academy { academyId: 1 }`
2. 쿠키 `activeAcademyId` 업데이트
3. 현재 페이지 리로드 → 선택된 지점 데이터로 표시
4. "전체 보기 모드" 선택 → `activeAcademyId = null` → 전체 집계 표시

| 메서드 | URL | 설명 | 권한 |
|--------|-----|------|------|

| GET | `/api/super/users` | 전 지점 관리자 목록 | SUPER_ADMIN |

```typescript
// web/src/middleware.ts
// /admin/super/* 접근 시 SUPER_ADMIN 역할 확인
export async function middleware(request: NextRequest) {
  if (request.nextUrl.pathname.startsWith('/admin/super')) {
    const ctx = await getAdminContextFromRequest(request)
    if (!ctx?.isSuperAdmin) {
      return NextResponse.redirect(new URL('/admin', request.url))
    }
  }

}
```

## 7. 검증 시나리오

| 시나리오 | 확인 포인트 |
|----------|-------------|
| 신규 소방학원 지점 생성 | Academy 레코드 + 기본 설정 + 시험 과목 자동 삽입 |
| 소방학원 관리자 로그인 | 경찰학원 학생 목록이 안 보임 |
| SUPER_ADMIN 지점 전환 | 전환 후 선택된 지점 데이터만 표시 |
| 소방학원 시험 과목 변경 | 경찰학원 성적 입력 화면에 영향 없음 |
| "전체 보기" 모드 | 모든 지점 합산 집계 표시 |
| 비활성화된 지점 | 해당 지점 관리자 로그인 시 접근 차단 |
